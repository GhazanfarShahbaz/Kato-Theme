## Installation

### From VS Code Marketplace
1. Open **Extensions** (Ctrl+Shift+X / Cmd+Shift+X)
2. Search for "Kato"
3. Click **Install**
4. Select **Kato** from the Color Theme picker (Ctrl+K Ctrl+T / Cmd+K Cmd+T)
